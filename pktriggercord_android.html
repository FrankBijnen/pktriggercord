PkTriggerCord Android port

The android port makes it possible to tether your camera using your mobile phone or tablet.

The android port of pktriggerCord is experimental. Use only at your own risk.

Requirements

  • Android 4.1 or newer device (tested only using 4.2, 5.1).
  • OTG USB port.
  • Root access.
  • Supported Pentax camera (check the list here).
  • A cable which connects the android device and a camera.

Features

Right now the program is very-very simple:

  • The program automatically connects to the camera (the bottom red line shows this info).
  • The program continuously updates basic info like: shutter speed, aperture,...
  • You can press the Focus button to focus.
  • You can press the Shutter to make pictures. You can specify the number of frames (left widget) and the delay between the frames (right widget - seconds).
  • The program shows the last picture and the progress of the exposure series.
  • The pictures are stored in /mnt/sdcard/pktriggercord. You can check the picture using RawDroid.
  • Output directory can be changed in Preferences
